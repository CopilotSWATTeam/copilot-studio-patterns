# Get File Icon

This is a simple *topic* that can take a file URL as an input and return a URL for an icon that represents the file type.

## 

## Topic Parameters

| Name           | Direction | Data Type | Notes                                                                                       |
|----------------|-----------|-----------|---------------------------------------------------------------------------------------------|
| **inFileUrl**  | Input     | String    | A URL to a file. The icon for the file is determined by its extension. Default is a *link*. |
| **outIconUrl** | Output    | String    | The icon for the file.         

                                                             |
